DOM

It's a tree data structure in which all HTML elements are organized in tree data structure and we can modify it through Javascript.

DOM can be modified through 2 ways

  1. Direct change thorugh HTML
  2. Change tbrough Javascript

DOM Methods

  1. getElementById
  2. getElementByClassName
  3. getElementByTagName
  4. querySelector
  5. querySelectorAll

innerHTML: is a property used to get or set the HTML content of an element. It allows you to update the content inside an HTML tag dynamically.

.innerText:is a property used to get or set the text content of an HTML element. It only returns the text content without any HTML tags.
